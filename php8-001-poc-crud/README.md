# Sistema CRUD de Usuarios
### Prueba de Concepto - PoC

Sitio web para la gestión de usuarios desarrollado con PHP 8 conexión PDO, MySQL, Bootstrap 5 y otras tecnologías.

## Objetivo

Desarrollar las principales lógicas para el CRUD:

- Conexión PDO con la base de datos MySQL: database/conexión.php *function conectarBD()*
- Proceso para guardar y editar desde el formulario: guardar.php *accion crear y editar*
- Proceso para eliminar registro: eliminar.php
- Proceso de búsqueda y paginación de la Lista de Usuarios: index.php *bloque de Configuración de paginación y Ajustar paginación..*

## Características

- ✅ CRUD completo (Crear, Leer, Actualizar, Eliminar)
- ✅ Diseño responsivo con Bootstrap 5
- ✅ Conexión PDO segura a MySQL
- ✅ Validación de formularios (cliente y servidor)
- ✅ Paginación y búsqueda
- ✅ Mensajes de confirmación y notificaciones
- ✅ Estadísticas en tiempo real
- ✅ Efectos visuales con Typewriter y animaciones CSS
- ✅ Código documentado y estructurado

## Requisitos del Sistema

- PHP 8.0 o superior
- MySQL 5.7 o superior
- Servidor web (Apache, Nginx, etc.)
- Extensión PDO habilitada en PHP
